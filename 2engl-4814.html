Developing Intelligent Content

Professor: Carlos Evia

What is Developing Intelligent Content?

I probably should not be publicly picking favorites, but this is the class I was most looking forward to over the summer and that did not change once classes had started. This is my first (and most entertaining) of three classes in Shanks 360. I started working with VS Code and GitHub about a year ago through my internship with the Cyber Range, so I was very excited to learn that we would be utilizing both these tools in class. This course is about learning how to create online content, and boy is it fun! We will spend the semester looking at markup language, content structuring, metadata, and the successful creation of adaptive content. By the end of the semester, I will have a resume in website form that I am very excited to create!
